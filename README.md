Go (struct) types, methods and interfaces. Based on
https://learning.oreilly.com/library/view/introducing-go/9781491941997/ch07.html. And some testing.

Usage:

```
cd /tmp
git clone git@github.com:jreisinger/multi-shape.git
cd multi-shape
export GOPATH=`pwd`
go test geom
go install calc
./bin/calc
```
